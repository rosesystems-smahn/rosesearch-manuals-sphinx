검색엔진 구동 및 정지
==============

<span></span>
1. 서비스 구동
------------
### 1.1. 검색엔진
```
$ fastcatsearch/bin/fastcatsearch start
```

### 1.2. 관리도구
```
$ fastcatsearch-console/start-console.sh
```

2. 서비스 정지
------------

### 1.1. 검색엔진
```
$ fastcatsearch/bin/fastcatsearch stop
```

### 1.2. 관리도구
```
$ fastcatsearch-console/stop-console.sh
```
